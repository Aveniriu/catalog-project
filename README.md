# Каталог часов

Тестовое задание: секция каталога интернет-магазина.

## Технологии
- HTML5
- CSS3 (CSS Variables, Grid, Flexbox)
- Адаптивная верстка (десктоп, планшет, мобильный)

## Как запустить
1. Открой index.html в браузере
2. Или используй Live Server в VS Code

## Особенности
- Модульная архитектура CSS через переменные
- Hover-эффекты на карточках и кнопках
- Полностью адаптивный дизайн
